# Shell
Custom Shell script made in C (to teach me C and kick start other projects)

Builtin Functions:
- cd
- help
- jobs
- fg
- bg
- alias
- unalias
- history
- exit

Supported Funcitonality:
- Pipes (|)
- Redirection (<, >, and >>)
- Enviornment variables ($) 
- Assignments (alias and EVs (=)
- Background Jobs (&)
- Ctrl+C & Ctrl+Z
- Arrow keys (History)
